RIVERPOD PATTERNS USED:

1. StateNotifier<State> - Mutable state logic
2. StateNotifierProvider - Expose notifier
3. Provider - Computed/read-only state
4. ConsumerWidget - Read single provider
5. ConsumerStatefulWidget - Read multiple providers
6. ProviderScope - Wrap entire app
